Завдання 7. Метод Монте-Карло для кидання двох кубиків
Опис реалізації

У цьому завданні реалізовано програму на Python, яка використовує метод Монте-Карло для моделювання процесу кидання двох гральних кубиків.

Програма:

імітує велику кількість кидків двох кубиків;

для кожного кидка обчислює суму значень;

підраховує кількість появ кожної можливої суми (від 2 до 12);

обчислює ймовірність кожної суми як відношення кількості її появ до загальної кількості симуляцій;

порівнює отримані результати з аналітичними ймовірностями;

візуалізує результати у вигляді таблиці та графіка.

Для генерації випадкових значень використовується стандартний модуль random.

Логіка алгоритму

1. Задається кількість симуляцій (наприклад, 200 000 кидків).

2. Для кожної симуляції випадково генеруються значення двох кубиків (від 1 до 6).

3. Обчислюється сума та зберігається у лічильнику.

4. Після завершення симуляції ймовірності обчислюються за формулою:

P(сума)= кількість появ суми​ / загальна кількість кидків

5. Отримані значення порівнюються з аналітичними результатами для двох кубиків.

Висновки

Метод Монте-Карло дозволяє ефективно наближати ймовірності випадкових подій шляхом статистичної симуляції. Реалізована програма підтверджує теоретичні розрахунки для задачі кидання двох кубиків.
